# MyStoreFullStack

Full-Stack E-Commerce Store

A Simple E-Commerce Store built with Angular Js, Node Js, Boostrap.

## Features

1. Cart System
2. Product List
3. Product Details
4. Checkout

## How to run on local

`npm install`

`npm start`
